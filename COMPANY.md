# Software Factory Method

Portable company package for running a BMAD-inspired software factory inside Paperclip.

This package is intentionally generic:

- no bundled client backlog
- no hardcoded repository binding
- no private runtime history
- no tenant-specific schedules or attachments

Attach your own project after import, then use the included agents and skills as the operating model.

This is an independent community package. It is not an official BMad Code or Paperclip company template.
